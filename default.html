p-string

length prefixed data string

by Charles Iliya Krempeaux

A p-string is a type of data string where there are 2 parts of the string:

  1. a length prefix, and
  2. the actual data of the string.

For example:

Name

The name “p-string” is named after the Pascal programming language. The “p” in “p-string” is short of “pascal”.

In fact, an alternative name for “p-string” is: pascal-string.

Although note that, the original Pascal programming language did not seem to have the p-string. The p-string seems to have been added to the Pascal programming language later. A data string seems to have been added to thePascal programming language due to being influenced by the BASIC programming language.

length prefix

With the original p-string, the length prefix was a single 8-bit byte. This was because of the limitations of the of the computer hardware of the era when the p-string was introduced to the Pascal programming language.

But, as the p-string was used in other places the length prefixed increased in size. And, of course, once the length prefix is more-than one byte in size, the length prefix also has an endianness — little-endian, big-endian, etc.
